# VoiceSpirit 记忆中心 (Memory Center) 使用指南

VoiceSpirit 集成了 EverMemOS，提供了一种创新的“长期记忆”体验。当您开启该功能后，VoiceSpirit 能够自动学习并记住您在各个场景中的互动内容，并在未来的对话或生成任务中主动利用这些背景知识。

## 什么内容会进入长期记忆？

只要您在设置中开启了对应的**场景开关**，以下内容将自动写入长期记忆：

- **AI 聊天 (Chat)**：您与 AI 助手对话的上下文、提到的偏好、任务目标以及关键事实。
- **语音聊天 (Voice Chat)**：在实时语音交互中交流的核心信息。
- **录音转写 (Audio Transcription)**：默认情况下，**全量录音文本不会包含在长期记忆中**。系统只会自动抽取并保存录音里提炼出的“关键信息摘要、待办事项和重点结论”等结构化记忆。如果你想保存全量原文用于后续无损检索，需额外在设置页开启“录音全量转写归档 (Store Full Transcript)”。
- **播客生成 (Podcast Script)**：您生成播客时提供的草稿或生成的对话脚本重点。
- **文本朗读 (TTS Synthesis)**：默认关闭。强制开启后，会对零碎朗读的文本抽取核心意图或偏好归档。

*注意：所有记忆的写入前提是您未开启“临时会话”模式，且对应的场景开关处于 ON 状态。*

## 什么不会默认进入长期记忆？

为了保护效率和隐私，以下内容在大都情况下不会默认进入长期记忆：

- **尚未选中的场景**：例如“文本朗读 (TTS Synthesis)”为了防止纯工具性朗读产生大量噪音，您可以决定是否在设置中手动开启。
- **大段冗余操作**：主要以用户价值和高信息密度的语义为主被抽取。

## “临时会话” (Temporary Session) 是什么意思？

开启“临时会话”后，VoiceSpirit 将进入类似浏览器的“无痕模式”：
- **不检索**：AI 在响应时不会检索您以前的任何长期记忆。
- **不写入**：在临时会话期间发生的任何内容，都不会被写入 EverMemOS 中。

如果您想讨论某些特定的、一次性的话题，或者只想进行一次纯净的生成测试，建议勾选该选项。

## 如何关闭或清理记忆？

- **临时关闭/彻底关闭记忆功能**：
  在“设置”页面中，随时可以取消勾选“开启 EverMemOS 长期记忆”选项，并且保存。关闭后，VoiceSpirit 与记忆中心的所有通讯都将停止。
  
- **关于清理记忆分身**：
  若要实现纯净的环境，可以通过后台重新注册并更换分配给您的 `EverMem API Key` 或者相应的 `Scope ID`，以此指向一个新的记忆库；或者按照 EverMem 管理台的教程清空目前库内节点。
